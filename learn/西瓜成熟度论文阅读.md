##评价指标
---
###成熟度的评价指标
==坚实度==
坚实度与糖度成负相关
坚实度越低越熟

| 声音 | 西瓜成熟度 |
| ---- | ---- |
| 咚咚咚（清脆） | 未熟 |
| 嘭嘭嘭（低浊） | 成熟 |
|  噗噗噗 | 过熟 |
